# Layout Rules

## Put in `SKILL.md`
- shortest usable workflow
- how to choose among bundled references
- non-obvious constraints that affect every invocation

## Put in `references/`
- long examples
- schema or module listings
- checklists
- variant-specific guidance

## Put in `scripts/`
- repeated deterministic transformations
- setup or sync helpers
- anything the agent would otherwise rewrite often
